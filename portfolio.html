🤖 GitHub-auto-labeler-bot + dashboard

A full-stack GitHub bot that auto-labels Issues and PRs using rule-based NLP and ML inference. Features a live dashboard to visualize labeling activity. Built with TypeScript, Express, MongoDB, and EJS. Modular, Dockerized, and CI/CD-ready.

  • Automation: Webhook listener triggers keyword/ML-based label engine
  • ML Inference: Supports zero-shot classification via Hugging Face models
  • Dashboard: EJS frontend with filters, live refresh, and activity logging
  • Persistence: MongoDB (LabelEvents) via Mongoose
  • DevOps: Docker Compose + GitHub Actions for local and CI/CD deployment

🔗 shrink-URL

A full-stack URL shortener with real-time analytics and a modern dashboard. Built with the MERN stack using TypeScript, it tracks click counts, access timestamps, and geolocation, simulating a production-grade SaaS tool like Bitly.

  • Frontend: React + Vite dashboard with auto-refresh and link management
  • Backend: TypeScript + Express REST API for shortening and analytics
  • Database: MongoDB Atlas with Mongoose for URL and metrics storage
  • Tracking: Records clicks, last access time, and location via IP API

📝 note-nest

An agentic AI pipeline for extracting and summarizing academic preprints from arXiv using LangGraph and FastAPI. Implements a multi-agent workflow with non-linear retry logic, LLM-powered summarization, and Redis-backed session state for intelligent academic search and synthesis.

  • Agent Workflows: LangGraph execution graph with conditional retry loops
  • Summarization: Hugging Face flan-t5-base for 3-sentence abstracts
  • API: High-performance async REST endpoints with FastAPI
  • State: Redis for user sessions, retries, and multi-turn context
  • Automation: Retriever, Summarizer, Evaluator & Compiler agents

📗 Goodreads-companion

A fully client-side Chrome Extension that enhances Goodreads with real-time metadata extraction, sentiment-aware review analysis, ZIP-based library lookup, and interactive data visualizations—no backend required. Demonstrates Manifest V3 architecture, NLP, structured data parsing, and stateful UX design.

  • Metadata: Parses embedded JSON-LD for title, author, ISBN
  • NLP: Classifies user reviews via lexicon-based sentiment analysis
  • Library Lookup: Queries Open Library API; falls back to WorldCat by ZIP
  • Visualization: Renders review sentiment using Plotly.js bar charts
  • State: Persists ZIP and settings via chrome.storage.sync
